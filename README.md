# Analyzing the behavior of a person recorded by a set of sensors
This is a Java 1.8 program that extracts some sensor data from a file and then processes it using Streams and shows statistical results based on the measured values.
#### The file that contains the sensor data( 
[Available online at](https://archive.ics.uci.edu/ml/datasets/Activities+of+Daily+Living+(ADLs)+Recognition+Using+Binary+Sensors)
